Root: `{}`

Python syntax is used for types.


`completion`: list[str]
-
> Indicates what storyline completion events have been accomplished in this save.
>
> For example, this could be `["tutorial","boss1","boss2"]` (placeholder bosses)

`maxCompletion`: int
-
> How many completion events exist. This is used for completion percentage.

`playtime`: int or float
-
> Total playtime on the save, in seconds.
